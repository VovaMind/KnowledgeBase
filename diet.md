# Диетические заметки

## Идеи

*    лучше скушать меньше, чем больше
*    не ешь, если уже сытый (ешь медленнее)
*    Молочка/овощи/фрукты/злаки хороши для микрофлоры желудка. Злаки: Гречка, рис, овес, кукуруза, пшено, рожь, пшеница
